#The Beginning ~ CMS

Tulisan-tulisan saya di blog bermula dari platform blog menggunakan CMS, saya sendiri dari dulu sangat prefer pada salah satu CMS yaitu joomla. Sering sewa hosting dan domain dari sejak duduk di bangku SMK dan aktif ngeblog, walaupun pada waktu itu selalu gonta-ganti blog dan banyak membahas topik yang gado-gado karena tidak terlalu serius untuk fokus membangun blog dimana pada akhirnya semua tulisan-tulisan pun hilang entah kemana. Sejak awal tahun 2020 saya memutuskan untuk fokus menulis hanya di satu blog dan mem-posting tulisan-tulisan saya di blog ini. Alasan dari semua ini karena kebetulan saya juga saat ini duduk di bangku perkuliahan dan menjadi mahasiswa dengan jurusan Teknik Komputer, oleh sebab itu bagi saya menulis di blog sangatlah penting karena bisa menjadi wadah dokumentasi saya setelah melakukan banyak hal yg telah saya pelajari khususnya di bidang ilmu IT.

#Saya jatuh cinta dengan ~ Hugo

Dari beberapa pengalaman saya ngeblog menggunakan CMS saya akhirnya menemukan cara yang tidak biasa dalam hal bloging, yaitu nge-blog menggunakan Static Site Generator dari Hugo. Hal yang tidak saya duga dari sekedar nge-blog menggunakan Hugo akhirnya saya dapatkan dengan semakin sering saya mengulik lebih dalam teknologi yang ada pada Hugo.


Entah mengapa saya jadi menyukai Programming, karena nge-blog menggunakan Hugo lebih banyak berhadapan dengan aplikasi text editor dan terminal akhirnya jiwa keinginan menjadi seorang programmer pun tumbuh "engga tau juga mau jadi programmer apa", selebihnya saya juga banyak berurusan di sisi BackEnd dengan GitHub Page dan Cloudflare "saya sih anggepnya gitu" . Semangat belajar Programming saya pun jadi tinggi dan mulai fokus saat ini mendalami static site Hugo dan mulai mengambil kursus pemrograman mobile Android menggunkana Bahasa Pemrograman Kotlin.


#Kenapa membuat Blog ini?

Alasan utama membangun blog ini hanya sebagai catatan pribadi tentang apa saja yang sudah saya pelajari dan saya praktekan, selebihnya saya menaruh harapan besar dimana tulisan saya ini akan bermanfaat untuk orang lain yang saat ini sedang menikmati proses belajar mengenai ilmu di bidang IT. Melalui blog ini saya juga ingin membagikan pengalaman dan ilmu saya "walaupun ilmu dan pengalaman juga engga banyak".


Topik yang akan sering saya bahas pada blog ini tidak akan jauh-jauh dari seputar Hugo, Devops, Programming dan selebihnya obrolan-obrolan pribadi saya mengenai IT.

#Saya ngapain? ~ Sekarang!

Saat ini saya bekerja di salah satu bagian divisi yang mengurus IT dari sebuah instansi Pemerintahan. Saya seorang pecinta Open Source dan pengguna Linux sejak tahun 2010.


Saya menyukai teknologi khususnya Open Source, menyukai kontribusi dengan teknologi yang di bangun menggunakan sistem Open Source, Linux Server Enthusiast, seneng baca buku dan nulis apapun di blog, suka bikin setup ruang kerja, suka kopi setangah pahit.


Saat ini saya pengguna setia Linux Debian dan MacOS, dan aktif melakukan eksperimen dengan server Linux serta mencari receh di internet.

#Perlu tulis Skill Saya disini?

Untuk saat ini skill yang saya miliki belum saya isi ya, saya masih ingin menguasai banyak kemampuan di bidang IT sebelum mengupdate form skill yang nantinya akan saya jadikan Portfolio di blog ini "maulu juga skill masih sedikit mau pamer".
