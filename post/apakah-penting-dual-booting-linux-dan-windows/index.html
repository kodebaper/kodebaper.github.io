Saya rasa penting tidak penting dual booting sistem operasi

Saya utarakan disini mengenai pendapat saya pribadi tentang penting atau tidaknya kita memutuskan untuk dual booting, misalkan kita memutuskan untuk sistem operasi Windows dan Linux terpasang di laptop/komputer kita. Disini saya mengutarakan dua jawaban yang bagi saya menjadi pertimbangan kamu apakah kamu benar-benar sudah mantap untuk hanya menggunakan jenis sistem operasi dari Linux atau masih berbarengan antara masih menggunakan Windows dan Linux.

Perlu di ketahui bahwa pemilihan sistem operasi bagi saya tidak ada yang sepenuhnya sempurna dan pastinya tanpa menjelek-jelekan salah satu jenis sistem operasi namun disini kita mesti terbuka dan mengakui bahwa sistem operasi yang kita gunakan memang harus sesuai dengan kebutuhan kita dan alangkah baiknya sisitem operasi yang kita gunakan juga sesuai dengan bidang pekerjaan kita.

Saya saat ini dual booting sistem operasi

Saya menggunakan Mac Os dan Linux ubuntu, jauh sebelum ini terjadi saya sebelumnya setia hanya memasang sistem operasi Ubuntu di Laptop. karena kebetulan alhamdulilah ada rejeki untuk membeli laptop baru dan akhirnya kebeli Macbook Pro dengan sistem operasi bawaan Mac OS Catalina saya tidak langsung menghapusnya karena untuk kebutuhan lain hal saya memerlukan Mac OS untuk kebutuhan seperti editing foto atau video dan yang tidak pernah terlupakan adalah Ubuntu yang selalu harus ada di perangkat laptop yang saya punya akhirnya saya memasang ubuntu di Macbook Pro saya. Karena hampir keseluruhan kebutuhan pekerjaan saya bisa di handle menggunakan Ubuntu seperti urusan pekerjaan Networking di kantor, testing server dan beberapa riset saya dalam hal development server semuanya menggunakan sistem operasi Ubuntu.

Mac OS tetap saya pertahankan karena memang saya khususkan untuk belajar editing foto dan video serta saya merasa untuk bisa juga mengoperasikan sistem operasi Mac OS, dan tidak hanya itu sebenarnya masih ada juga sistem operasi yang saya pasang menggunakan Virtual Machine di atas sistem operasi Mac OS saya yaitu Kali LInux untuk mulai sedikit-sedikit belajar Security serta CentOS untuk belajar membangun server karena saya pikir untuk fokus belajar server hanya menggunakan sistem operasi Ubuntu tidaklah puas bagi saya sendiri sehingga saya mesti ada pengalaman untuk praktek membangun sebuah server dengan menggunakan sistem operasi Centos.

Kesimpulan !

Seperti pembahasan saya di awal artikel ini, bahwa kita mesti menyesuaikan dalam hal pemilihan sistem operasi yang kita gunakan, salah satu contoh jika kamu cukup dengan menggunakan sistem operasi Ubuntu untuk keperluan pekerjaan Kamu, mengapa kamu mesti memberatkan kinerja Laptop/PC kamu dengan install dual booting atau lebih ? dan yang tak kalah penting adalah kita mesti menghormati dan menghargai karya seseorang dengan menggunakan sistem operasi yang legal, Kamu banyak uang ? silahkan menggunakan sistem operasi berbayar dan gunakan lisensi resmi. Dan jika kamu belum cukup uang untuk dapat membeli lisensi sebuah aplikasi atau sistem operasi kenapa mesti menggunakan produk bajakan sedangkan alternatif beralih ke produk Open Source sangat banyak.

Semoga artikel ini dapat membuka wawasan kita untuk menggunakan perangkat teknologi software yang legal.
