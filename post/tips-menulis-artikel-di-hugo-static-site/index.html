Sebagai seorang blogger yang menggunakan static site Hugo sebagai mesin blognya, ngeblog dengan terminal merupakan suatu hal yang cukup berbeda bagi saya karena sebelumnya dari dulu saya selalu membuat blog menggunakan CMS yang kaya akan fitur graphical user interfacenya. Namun kali ini saya dihadapkan dengan keadaan yang berbeda dimana menulis artikel di blog menggunakan Hugo cukup menggunakan text editor saja. Artikel ini akan saya update dengan pembahasan yang lebih lengkap lagi di lain waktu dengan tips-tips yang lain mengenai cara penulisan artikel di Hugo, adapun tips yang saya tulis saat ini masih sebatas keseluruhan yang saya pakai dalam penulisan di blog saya saat ini.

Untuk mengingatkan kembali, sebelumnya keseluruhan file Artikel atau tulisan kita di blog hugo akan di simpan di direktori content dengan urutan filenya seperti ini

    [Hugo blog]
                ---> [content]
                          |
                     [blog/post]

Adapun cara penulisan artikel pada blog yang menggunakan hugo/membuat artikel melalui text editor adalah sebagai berikut:

Memasukan file gambar

Saya biasanya menggunakan script ini apabila kita ingin memasukan gambar pada sebuah postingan artikel

<img src="/images/namagambar.png" align="center">

kita tinggal sesuaikan untuk file gambar di letakan dimana untuk selanjutnya kita panggil dengan menggunakan script ini, biasanya pada folder template akan di simpan di folder static lalu folder images tempat menyimpan aset gambar.

Membuat form script text

Saya menyebutnya seperti itu, karena jujur saya masih susah dan memang belum menemukan artikel yang membahas mengenai penulisan artikel di Hugo. Oke jadi ini kita gunakan untuk membuat tulisan berada di dalam form hitam, biasanya di dalam artikel ada kalanya kita ingin menulis script diantara tulisan di artikel tersebut, nah kita perlu membuatnya agar tulisan script ini di bedakan dengan tulisan biasa, untuk lebih jelasnya bisa lihat gambar di bawah ini.

Terlihat ada tulisan di dalam form hitam tersebut bukan ? dan ternyata semakin kita menulis panjang dan banyak akan menciptakan Scrollbar di dalam form hitam, dan ini sangat menarik. Lalu untuk membuatnya bagaimana ? Ternyata untuk membuatnya pun sangat mudah, kita cukup majukan tulisan/script yang akan kita masukan kedalam form hitam tersebut dengan cara menekan tombol Tab 2x lalu tulis atau paste tulisannya.

Membuat Garis di depan text

Nah untuk yang satu ini kita biasanya membuat tulisan di salah satu atau dua kalimat di tandai, untuk lebih jelasnya bisa dilihat dari gambar dibawah ini.

Dari gambar diatas ada garis lurus berwarna ungu bukan ? nah ini saya gunakan untuk menandai tulisan tertentu dalam sebuah kalimat di dalam artikel, kalau saya sendiri menggunakan penanda ini untuk meletakan tulisan yang berisi kalimat yang akan kita jelaskan di dalam artikel yang sedang saya bahas, agar lebih mudah di pahami. Karena bagi saya dengan menandainya maka akan ada penekanan pada kalimat tersebut.

adapun untuk membuatnya juga sangat mudah, kita hanya perlu memberikan 2x tab pada baris yang akan di tuliskan kalimat yang akan di beri garis dan berilah tanda > sebelum kalimatnya, untuk lebih jelasnya lihatlah gambar dibawah ini.

Sangat sederhana cara membuatnya, namun cukup penting disini peranannya.

Menandai tulisan dengan titik bulat

Untuk yang satu ini biasanya saya menggunakannya untuk mendeskripsikan banyak sub-sub judul, misalnya saya sedang mendeskripsikan jenis-jenis sistem operasi linux Debian dan turunanya, maka saya akan menyebutkan sistem operasi Debian tersebut dengan mengawali [TITIK BULAT] sebelum nama sistem operasi, contohnya adalah sebagai berikut.

  • Linux Ubuntu

  • Linux Mint

  • Kali Linux

  • dsb

Bisa di lihat bukan, titik bulat sebelum kata “Linux” ? nah maksud saya seperti itu. Bagi bloger yang menggunakan Hugo dalam penulisan artikelnya saya rasa untuk pemula seperti saya hanya memberikan titik bulat akan cukup kesulitan, untuk itu disini penting tidak penting tetap saya bahas toh saya juga menggunakannya.

Untuk membuatnya sangatlah mudah, kita cukup memberikan tanda [-] sebelum kalimat yang akan kita berikan tanda titik bulat, untuk lebih mudahnya bisa lihat gambar di bawah ini.

Membuat judul bergaris

Pada fungsi ini saya akan memberikan tips bagaimana membuat judul tulisan besar bergaris pada artikel atau sub judul dalam setiap postingan. Untuk lebih jelasnya misalnya seperti ini, saya menyematkan sub-sub judul di setiap artikel.

contoh sub judul 1 bergaris

contoh sub judul 2 bergaris

Seperti ini, adapun untuk membuatnya juga simple yaitu memberikan tanda [##] di setiap judul yang di inginkan. untuk lebih jelasnya perhatikan gambar dibawah ini.

Namun jika kita tidak menginginkan adanya garis di bawah judul, maka kita tinggal menghapus satu tanda # sebelum judulnya. Demikian untuk artikel ini saya cukupkan sampai disini dan kedepan akan saya update tips-tips yang lain mengenai cara penulisan artikel di blog yang menggunakan Hugo ssg. Semoga tulisan ini bermanfaat, selamat belajar!
