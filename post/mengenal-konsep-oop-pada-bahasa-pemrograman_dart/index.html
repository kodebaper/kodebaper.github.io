Programming Dart Sesi OOP

Yuk mengenal pondasi dasar yang sangat penting sebelum melangkah untuk mempelajari dari bahasa pemrograman Dart, hal pertama yang saya mesti pelajari adalah belajar dari hal yang paling dasar yaitu OOP dari bahasa pemrograman Dart itu sendiri. Lalu apa it OOP?

Object Oriented Programming adalah suatu teknik dalam pemrograman yang berdasarkan konsep “Object” dimana Object tersebut dapat berisi data, dalam bentuk variabel atau dikenal juga sebagai atribute, serta fungsi/prosedur atau dikenal juga sebagai method.

inti dari pembelajaraan ini saya akan membahas 6 pokok pembahasan, yang diantaranya yaitu:

  1. Object
  2. Class
  3. Inheritance
  4. Polymorphism
  5. Interfaces
  6. Abstract Class

  • Apa itu Object?

    Adalah suatu abstraksi yang dimaksud untuk menyajikan komponen dari program. Dimana Object ini dapat di ciptakan, dapat dihapus, dapat diberikan atribut dan dapat melakukan suatu aksi.

    Mudahnya kalau saya sendiri memahami Object ini adalah dimana komponen dari suatu program yang dapat di ciptakan dihapus atau ditambahkan yang dapat melakukan suatu aksi tertentu nah si program ini di buat di dalam suatu alur menggunakan prosedur/method.

  • Apa itu Class?

    Secara singkat Class dapat diartikan sebagai sebuah blueprint dari suatu Object, dimana kita mendesain suatu Object berdasarkan Class yang kita buat, saya contohkan di dalam dunia nyata, contohnya jika kita ingin membuat suatu rumah tentu kita memerlukan desain dari rancangan rumah tersebut, detali denah ruangan, ukuran serta ruang apa saja yang akan di include di dalam desain rumah tersebut. Lebih mendetail lagi kita akan membuat desain barang apa saja nantinya yang akan kita letakan di rumah tersebut di setiap ruangannya.

    Dapat saya simpulkan bahwa Class sendiri merupakan hasil dari sebuah rancangan dan hasil dari rancangan itu berupa objeknya, yang kalau kita tarik dari analogi saya diatas yaitu, rancangan berupa detail denah dari sebuah rumah lalu hasil dari rancangan tersebut berupa rumah dan dengan segala jenis ruangan dan beserta isinya.

  • Apa itu Inheritance?

    Inheritance atau pewarisan ini mengadopsi dunia nyata dimana suatu entitas/objek dapat mempunyai entitas/objek turunan, ini dimaksudkan agar suatu Class dapat mempunyai Class turunan. Nah lalu mengapa perlu adanya Inheritance dalam sebuah konsep OOP?

    Jawabanya adalah kita perlu menerapkan Inheritance pada saat kita menjumpai ada suatu Class yang perlu perluasan Class lain yang sudah ada.

    Kemudian pada bahasa pemrograman Dart itu sendiri mendukung adanya pewarisan, yang digunakan untuk membuat Class baru dan dari Class yang sudah ada. Class yang akan di-extend disebut parent/superClass**, dan *Class* yang baru dibuat disebut child/sub*Class*. *Dart* menyediakan kata kunci extends untuk mewarisi properti kelas induk di kelas anak.

  • Apa itu Polymorphism?

    Cukup membingungkan saya memahami istilah ini dalam bahasa pemrograman Dart, mudahnya saya memahami Polymorphism adalah mengacu pada pengertian ini. Dimana Polymorphism dapat diartikan dalam konsep OOP adalah sebuah metode agar beberapa Object dapat memiliki beberapa bentuk, kemudian Polymorphism sendiri merupakan suatu nama tunggal dengan menyatakan Object dari Class - Class yang berbeda yang terhubung pada satu superClass

  • Apa itu Interface?

    Dalam bahasa pemrograman Dart, Interface mendefinisikan sintaks yang harus di patuhi dalam setiap entitas. Interface menentukan sekumpulan metode yang tersedia pada suatu Object. Dart tidak memiliki sintaks untuk mendeklarasikan interface. Deklarasi Class sendiri merupakan Interface di Dart.

  • Apa itu Abstract Class?

    Abstract Class adalah Class yang masih dalam bentuk abstract, karena bentuknya masih abstract dia tidak bisa dibuat langsung menjadi Object. Lebih mudahnya memahami analogi abstract pada konsep OOP adalah saat kita menyebutkan kata “KENDARAAN” tentu saat kita definisikan lebih detail akan timbul pemahaman yang begitu banyak pula, kendaraan yang dimaksud bisa berupa kendaraan berupa transportasi darat, laut, ataupun udara, belum lagi dia berjalan atau terbang? bentuknya dalam bentuk seperti apa?

    Tentu ini akan sangat banyak definisi bukan? ini yang di sebut masih abstrak karena kita belum menyebutkannya secara spesifik.
