Distro Linux yang cocok untuk Pemula

Pada artikel di season 1 saya telah membahas 5 point penting untuk seorang pemula setelah memutuskan untuk bermigrasi ke sistem operasi open source, nah pada kesempatan ini saya akan mulai membahas point pertama. jika dibaca pada artikel pertama yang saya bahas di season 1 sengaja saya tidak jabarkan terlalu panjang karena saya berasumsi bahwa artikel yang terlalu panjang lebar yang di sajikan pada satu post artikel akan membuat rasa malas pembacanya.

Karena bagi saya sendiri melihat artikel yang terlalu panjang sudah membawa kesan minat membacanya yang terlanjur malas dan maunya artikel yang di baca langsung ke inti pembahasan, itu sebabnya saya membagi per season agar artikel lebih pendek dengan harapan artikel yang di tulis tidak terlalu panjang akan lebih membuat efektif orang yang akan membawanya untuk membaca keseluruhan dari isi artikel namun tanpa mengurangi kualitas dari point penting pembahasan topik artikelnya.

Lantas distro linux apa sih yang cocok untuk seorang pemula yang baru migrasi ke sistem operasi Open Source Linux Versi saya sendiri?

Distro linux rekomendasi pilihan saya jatuh pada ubuntu, dimana ubuntu saat ini sudah semakin bertransformasi untuk menghadirkan lingkungan antar muka yang tidak kalah dengan sistem operasi lain seperti MacOS dan Windows, terutama dalam hal kemudahan penggunaanya. Itu sebabnya linux ubuntu sangat cocok untuk di gunakan seorang pemula. Adapun keunggulan dari sistem operasi ubuntu diantaranya yaitu:

Software Center yang lengkap

Kamu Pengguna Linux Ubuntu tidak perlu khawatir soal kebutuhan akan software, karena di ubuntu ada Software Center yang siap untuk memenuhi kebutuhan aplikasi, tinggal memilih kategori aplikasi lalu instal sesuai kebutuhan.

Relative kecil kemungkinan terkena Virus

Keunggulan berikutnya dari sistem operasi GNU Linux adalah sedikit kemungkinan terkenanya virus pada sistem kita, karena setau saya begitu jarang mendengar ada sistem operasi linux terkena virus. Pengalam saya sendiri hampir kurang lebih 10 tahun menggunakan linux tidak pernah mengalami kendala sistem operasi linux yang saya install di pc maupun di laptop terkena virus. Namun yang menjadi pertanyaan adalah apabila di misalkan sebuah virus dari sistem operasi Windows di inveksikan ke sitem operasi Linux, apakah bisa ?

Jawaban saya adalah tidak bisa, karena keduanya sistem operasi ini bekerja sangat berbeda. Dengan begitu, virus yang mencari C:\Windows\System32folder kmau di windows tidak akan menemukannya di Ubuntu karena tidak ada.

Lingkungan Desktop yang bagus dan mudah di kustomisasi

Kamu sebelum pindah ke sistem operasi linux adalah pengguna setia Windows ? jika ia, maka kamu sangat di sarankan untuk menggunakan linux ubuntu untuk menjadi sistem operasi pertama kamu saat memilih untuk migrasi sistem operasi. Kenapa ?

Jawabanya karena kamu tidak akan kesulitan untuk menyesuaikan dengan linugkungan antar muka yang di suguhkan oleh sistem operasi Linux ubuntu, banyak pilihan desktop environment yang bisa kita costumize dengan mudah di sistem operasi ubuntu, bahkan merubah tampilan ubuntu kita mirip dengan sistem operasi windows ataupun Mac OS sangatlah mudah.

Memulai belajar perintah dasar Terminal dari sistem operasi Ubuntu

Apa alasannya ? apa mesti pemula harus belajar perintah dasar terminal dari sistem operasi ubuntu ?

Tidak juga, buat kamu mungkin atau yang lain, karena ini merupakan pendapat dan sepengalaman saya sendiri saat ini mengenali dan sedikit banyak memahami penggunaan perintah pada sistem operasi turunan dari Debian, turunan dari RedHat, dan freeBSD di masing-masing sistem operasi dan turunanya antara Debian, RedHat atau sistem operasi freeBSD akan sangat berbeda dalam hal penggunaan perintah CLI. Saya sedikit memahami perintah-perintah dari sistem operasi tersebut itu semua berkat modal awal saya belajar perintah dasar dan membiasakan dengan perintah-perintah dasar terminal dengan menggunakan sistem operasi Linux Ubuntu.

Salah satu contohnya

Kamu akan menggunakan perintah update paket di sistem operasi `Debian` dan 
turunanya dengan perintah
    # sudo apt-get update

Namun Kamu tidak akan bisa mengupdate paket di sistem operasi `RedHat` dan
turunanya, namun menggunakan perintah update
    # yum update

Ini merupakan contoh kecil, dan akan berbeda juga untuk beberapa fungsi dari perintah di setiap masing-masing sistem operasi dan berlaku untuk turunan dari sistem operasi tersebut juga.

Loh mengapa mesti Linux mint ? bukannya sama saja turunan dari sistem operasi Debian ?

Sekali lagi, disini saya bahas dari sepengalaman saya ya. karena Linux Mint merupakan sistem operasi kedua yang saya pelajari setelah mulai terbiasa dengan Ubuntu. Lalu apa yang saya dapat ?

Saya akan menemukan fungsi dari perintah Terminal pada Ubuntu atau Linux Mint sama, dan lingkungan antar muka juga yang relatif sama, namun Linux Mint bagi saya dapat menghadirkan kesan yang berbeda dan menambah pemahaman kita akan penggunaan dari sistem operasi berbasis Linux. Karena makin berfariasinya pilihan desktop enviroment pada Linux Mint seperti contohnya DE Cinnamon, ataupun Mate dimana tampilan dari Linux Mint sendiri yang semakin sempurna untuk mobilitas selama penggunaan untuk melakukan tugas pekerjaan sehari-hari.

Dari pengalaman menggunakan Ubuntu lalu Linux Mint saya pelan-pelan memilik sedikit modal pengetahuan untuk mulai menggunakan dan mempelajari sistem operasi berbeda turunan dari Debian dimana saya mulai belajar menggunakan Fedora dan mulai berkenalan dengan freeBSD hingga saat ini.

Kedua sistem operasi yang saya bahas di artikel ini, merupakan sistem operasi sepengalam saya dari dulu awal menggunakannya dan saya rasa patut di jadikan pertimbangan untuk menjadi sistem operasi linux pilihan pemula, karena saya dulu mengawalinya juga dari seorang pemula. Yah walaupun sekarang juga tidak begitu jago-jago amat. Semoga artikel ini bermanfaat dan selamat belajar!
