Cara Terbaru Memasang Disqus Comment Di Blog Hugo Static

Berikut merupakan cara memasang Disqus Comment yang saya terapkan di blog saya kali ini. Cara ini sedikit berbeda dengan tutorial yang saya tulis sebelumnya di blog ini juga, adapun cara yang saya terapkan saat ini lebih simpel dari cara sebelumnya.

Berikut merupakan cara memasang disqus comment di blog hugo static.

  1. Saat ini saya angga semuanya sudah memiliki account di disqus ya.

  2. Ikuti seluruh petunjuk penginstallan disqus comment sampai ke tahapan dimana kamu diberikan code dari disqus, code tersebut kurang lebihnya seperti ini.

<div id="disqus_thread"></div>
<script>
    /**
    *  RECOMMENDED CONFIGURATION VARIABLES: EDIT AND UNCOMMENT THE SECTION BELOW TO INSERT DYNAMIC VALUES FROM YOUR PLATFORM OR CMS.
    *  LEARN WHY DEFINING THESE VARIABLES IS IMPORTANT: https://disqus.com/admin/universalcode/#configuration-variables    */
    /*
    var disqus_config = function () {
    this.page.url = PAGE_URL;  // Replace PAGE_URL with your page's canonical URL variable
    this.page.identifier = PAGE_IDENTIFIER; // Replace PAGE_IDENTIFIER with your page's unique identifier variable
    };
    */
    (function() { // DON'T EDIT BELOW THIS LINE
    var d = document, s = d.createElement('script');
    s.src = 'https://yourblog.disqus.com/embed.js';
    s.setAttribute('data-timestamp', +new Date());
    (d.head || d.body).appendChild(s);
    })();
</script>
<noscript>Please enable JavaScript to view the <a href="https://disqus.com/?ref_noscript">comments powered by Disqus.</a></noscript>

  1. Script diatas merupakan script yang di berikan oleh disqus sewaktu kita registrasi dan mengikuti tahapan pemasangan.

  2. Langkah yang paling penting pada tahapan ini, yaitu kita tidak perlu membuat file disqus.html di folder /partials blog kita.

  3. Kemudian ambil code dari disqus yang di berikan, lalu copy-paste kan pada file post.html pada folder /partials blog kita. File ini tidak selalu bernama post.html di setiap blog maka sesuaikan dengan nama file dengan penamaan file untuk menampilkan artikel pada postingan kita di blog.

  4. Pastikan code di paste di bagian paling bawah dari file post.html ya.

Hanya itu saja, kemudian tinggal kita isikan shortname pada file config.toml blog kita. Seharusnya setelah tahapan ini dilakukan dan kita deploy disqus comment sudah bisa tampil di bagian paling bawah artikel blog kita.

Bagi saya cara ini lebih simpel dan memudahkan penerapan fitur disqus comment di blog hugo static. Selamat mencoba dan semoga bermanfaat ya.
