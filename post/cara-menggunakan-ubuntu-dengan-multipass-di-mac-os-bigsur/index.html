Cara Menggunakan Ubuntu Dengan Multipass Di Mac OS BigSur

Beberapa waktu yang lalu, saya menemukan cara terbaru untuk tetap bisa menggunakan sistem operasi ubuntu pada perangkat MacBook Pro saya yang terpasang sistem operasi Mac OS, tanpa menggunakan VirtualBox atau VMware seperti yang biasa saya lakukan untuk meng-install sistem operasi Linux Ubuntu di dalamnya.

Lalu caranya bagimana ?

Saya hanya perlu menggunakan Multipass untuk dapat menggunakan sistem operasi ubuntu, kemudian cukup dengan baris perintah menggunakan aplikasi Terminal default bawaan dari sistem operasi Mac Os. Simpel bukan ?

Sebelum melangkah lebih jauh, saya akan menjelaskan sedikit apa itu teknologi Multipass. Penjelasan ini saya ambil langsung dari GitHub canonical/multipass untuk kita dapat memahami lebih dalam lagi.

Multipass is a lightweight VM manager for Linux, Windows and macOS. It's
designed for developers who want a fresh Ubuntu environment with a single command. 
It uses KVM on Linux, Hyper-V on Windows and HyperKit on macOS to run the VM with minimal 
overhead. It can also use VirtualBox on Windows and macOS. 
Multipass will fetch images for you and keep them up to date.

Since it supports metadata for cloud-init, you can simulate a small cloud deployment 
on your laptop or workstation.

Untuk langkah ini saya praktekan menggunakan sistem operasi MacOS, jadi untuk pengguna Windows atau Linux yang ingin menggunakan Multipass mungkin bisa mengikuti panduan dari GitHub canonical/multipass Disini

Oke mari ke langkah instalasi Multipass di sistem operasi MacOS BigSur.

Langkah paling awal buka aplikasi Terminal bawaan pada sistem operasi MacOS kita.

Saya memasang multipass lebih suka menggunakan Homebrew jadi perintahnya seperti ini.

$brew install --cask multipass

Kemudian cek ketersediaan images setelah kita berhasil meng-install Multipass nya. Dan ini hasilnya pada perangkat saya.

18.04                       bionic            20210512         Ubuntu 18.04 LTS
20.04                       focal,lts         20210510         Ubuntu 20.04 LTS
20.10                       groovy            20210511.1       Ubuntu 20.10

Kemudian saatnya saya meluncurkan Ubuntu versi terbarunya dengan perintah dibawah ini. Apabila sampai langkah ini tidak mengalami kendala harusnya hasilnya akan seperti dibawah ini

$ multipass launch ubuntu

Saatnya kita cek ketersediaan images yang telah selesai di download seperti langkah di atas ini. cukup ketikan baris perintah di bawah ini dan mestinya akan menampilkan pesan seperti ini.

multipass list
Name                    State             IPv4             Image
master-butterfish       Running           192.168.64.2     Ubuntu 20.04 LTS

Bisa di ketahui dari info seperti diatas ini bahwa image yang terpasang adalah ubuntu versi 20.04 LTS dengan IP Address yang terdeteksi 192.168.64.2 dengan status sedang berjalan.

Oke, mari kita lanjutkan lagi praktenya……

Saatnya kita ketahui lebih lanjut mengenai instance VM yang baru saja kita luncurkan dengan perintah ini. Seharusnya akan menampilkan informasi lengkap seperti di bawah ini.

Sampai dengan tahapan ini kita sudah berhasil meluncurkan ubuntu dengan multipass dan kita dapat menggunakan sistem operasi Linux Ubuntu nya cukup dengan aplikasi terminal bawaan kita di sistem operasi Mac OS
