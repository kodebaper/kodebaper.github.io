Ngobrolin apa di Kodebaper ?

Pada kesempatan kali saya akan membahas lebih jauh mengenai isi dari pembahsan utama blog ini. Lebih tepatnya saya menjelaskan mengenai apa saja sebenarnya isi dari konten blog saya, oke saya mulai dari pembahasan pertama dari blog ini ya.

DevOps

DevOps adalah serangkaian pekerjaan yang mengotomatisasi proses antara Software Development dan Development Team agar mereka dapat melakukan proses build, test dan release perangkat lunak lebih cepat dan lebih handal. {source:Medium.com/@alfianmalik}.

Dari penjelasan mengenai DevOps ini saja saya langsung tertarik untuk mengulik lebih dalam ilmu-ilmu apa saja yang harus di kuasai dan skill apa saja yang harus di miliki oleh seorang DevOps engineer, nah disini saya akan mendokumentasikan mengenai apa saja yang saya pelajari tentang DevOps.

Hugo

Di sini saya mengupas lebih dalam mengenai pengetahuan saya menggunakan hugo, berawal dari migrasi blog saya dari CMS ke Static Site Hugo maka hadirlah blog kodebaper ini. Dari migrasi beralih menjadi hal yang menyenangkan untuk setiap dokumentasi belajar yang saya lakukan dengan rutin menuliskan artikel di blog ini, Hugo membuat ketertarikan saya dalam dunia blogging menjadi semakin mendalam, karena dengan Hugo selain tujuan utama saya untuk menuliskan dokumentasi ilmu yang telah saya pelajari namun juga saya mulai memikirkan bagaimana memiliki sebuah blog yang powerfull dan menyajikan isi content di blog yang berkualitas, bukan dari segi isi artikel namun juga dari sisi blog itu sendiri menampilkannya untuk di baca seseorang menjadi lebih menyenangkan.

Linux Enthusiast

Pembahasan linux ini saya bukan seorang expert di bidang sistem operasi Linux, saya hanya pengguna Linux dari tahun 2010 yang saat ini masih belajar, di sini saya menuangkan seluruh/sebagian ilmu yang saya pelajari selama menggunakan sistem operasi, lebih tepatnya catatan saya udah ngapain aja sih make LINUX dari tahun 2010 dan karena ketertarikan saya dengan dunia server yang di bangun menggunakan sistem operasi Linux, makan saya menghadirkan pembahasan seputar Linux di blog ini. Intinya apa saja yang saya lakukan dengan sistem operasi Linux, saya berusaha akan menuangkannya dalam sebuah artikel di blog ini.

Pemrograman

Saat ini ketertarikan saya pada dunia programming sangat besar, terutama pemrograman untuk membuat aplikasi mobile pada device android. Apalagi dari segi benefit yang bisa di dapatkan jika mempunyai kemampuan menjadi mobile software engineer . Tentunya pointnya bukan ini, namun karena ketertarikan saya belajar pemrograman berbasis mobile untuk bisa menciptakan aplikasi yang dapat bermanfaat untuk kehidupan orang banyak [cita-citanya gini sih]

Secangkir Kopi

Tempat ngobrol ringan, disini saya membahas apa saja yang ada keterkaitannya dengan dunia IT. Saya tidak pandai bicara di depan orang banyak, saya ingin ngeGibahin apa saja yang saya mau dan saya tau mengenai IT dan juga perkembangan terbaru ataupun dalam kehidupan saya saya sehari-hari, maka saya ambil kesimpulan untuk menuangkan kata-kata saya dalam bentuk tulisan di sini. Maka disinilah tempatnya, intinya saya tuangkan dari obrolan Es Teh sampai obrilan Kacang Goreng
