Cara Membuat About Page Di Hugo SSG

Beberapa waktu yang lalu saya sempat mengalami kesulitan saat membuat halaman about atau sering di sebut about page, saya mengalami kesulitan karena memang baru pertama kali mempelajari dunia programming sederhana dari praktek yang juga sederhana dengan studi kasus untuk membuat beberapa halaman static di blog dengan static site generator dari Hugo.

Dan akhirnya saya menemukan beberapa langkah sederhana dan cukup mudah untuk membuat halaman di blog dengan menggunakan Hugo ssg, adapun cara yang saya dapat untuk membuat about page di hugo ssg yaitu sebagai berikut:

  1. yang paling pertama kita mesti membuat about.md dimana file ini akan kita panggil yang nantinya akan di tampilkan pada halaman about, untuk membuatnya cukup ketikan perintah ini

     hugo new about.md

    Setelah kita buat, mari kita perikas file about.md yang baru kita buat di folder Content pada file blog hugo kita. Untuk lebih jelasnya lihat gambar di bawah ini.

    Seperti yang kita lihat pada gambar di atas ini, di dalam folder /Blog, ada file about.md bukan ? ini berlaku sama, artinya jika kita ingin membuat halaman lain misalnya halaman contact, halaman FaQ, dll saya cukup membuat file .md layaknya file about.md yang tadi saya buat. Sampai sini ada gambaran ?

    Oke mari kita lanjutkan, setelah file about.md kita buat, mari kita lihat isinya. disini sudah saya isi, tinggal kalian sesuaikan dengan kebutuhan untuk isi dari halaman about page nya.

    Saya menambahkan beberapa baris kode Html, css, bootstrap, dll untuk kebutuhan isi dari halaman about agar tampak lebih menarik dan disesuaikan dengan layout halaman sesuai keinginan saya.

    sekarang mari kita kupas satu persatu dari isi file about.md ya.

     title: "Tentang Kodebaper"
 layout: staticpage
 draft: false

    title: "Tentang Kodebaper" ini akan kita gunakan sebagai file judul dari halaman yang dimana saat halaman about page ini di akses maka akan akan muncul.

    layout: staticpage ini adalah sebuah halaman HTML yang nantinya akan saya beri nama staticpage.html yang akan di gunakan untuk memanggil file about.md lebih jelasnya akan saya kupas di langkah ke 2

    draft: false ini adalah deklarasi agar file about page tidak tampil pada halaman post dan hanya akan tampil pada page about saja nantinya, ini berbeda dengan artikel yang mesti kita isikan dengan menyebutkan draft: true agar tampil pada halaman postingan. “koreksi jika penjelasan saya salah pada kolom komentar ya”

    Mari kita akan memasuki langkah ke dua.

  2. sekarang kita akan membuat file staticpage.html di dalam folde Theme utama blog hugo kita dimana file staticpage.html akan saya letakan di dalam folder _default dengan urutan seperti ini

     theme--
     |
     namaTheme--
         |			
         layouts--
                 |
                 _default
                     |
                     staticpage.html

    ingat, untuk penamaan file HTML staticpage.html harus sama dengan nama layout: staticpage pada file about.md di atas, untuk lebih jelasnya dalam peletakan file staticpage.html bisa lihat gambar di bawah ini.

    Bisa di lihat bukan, adal file staticpage.html di dalam folder _default . Nah sekarang mari kita lihat isi dari file staticpage.html untuk kemudian kita kupas isi dari file tersebut.

    adapun untuk penjelasan dari isi file staticpage.html yaitu sebagai berikut:

    Untuk mempersingkat penjelasan, akan saya jelaskan secara global untuk mempersingkatnya, disini terdapat beberapa baris kode seperti {{ partial “head.html” . }} {{ partial “header.html” . }} dan {{ partial “footer.html” . }}

    maksudnya adalah, ini akan memanggil file HTML dari head, header, dan footer pada folder /Partials di theme hugo kita, supaya halaman about nantinya akan tampil secara utuh sesuai tampilan blog dimana di dalam layout blog tersusun dari header sampai footer secara berurutan dan akan menjadi tampilan yang tidak hanya halaman polos dimana saat kita buka halaman about tidak hanya tampilan dari isi about page saja.

    Ada gambaran sampi di sini ? untuk lebih pahamnya bisa di coba-coba sendiri ya langsung praktek agar mengerti.

    Kemudian akan kita jumpai baris kode <section> {{ .Content }} </section> dimana baris kode ini akan memanggil dan menampilkan isi dari file about.md yang kita buat di awal.

  3. sekarang kita sudah selesai membuat halaman about page, dan kita saatnya menghubungkan dengan link about tersebut ke menu yang akan membuka about page. disini saya membuat menu about jadi link yang saya isi pada menu about seperti ini https://kodebaper.com/about

Akihrnya kita sampai di penghujung praktek dari tutorial membuat about page pada blog hugo, mudah bukan untuk membuatnya, selamat mencoba ya!
