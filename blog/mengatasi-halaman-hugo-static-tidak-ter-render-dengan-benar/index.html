halaman Hugo tidak di render dengan benar

Setelah beberapa waktu saya tidak meng update artikel yang membahas error di Hugo, kali ini akan ada pembahasan baru yang akan saya bagi terkait error yang saya alami beberapa waktu lalu. Dimana error ini penyelesaiannya bisa dikatakan sangatlah mudah, disini saya hanya perlu merubah baris perintah untuk permasalahan ini. Saya baru saja mendapati jika menggunakan perintah hugo server untuk menjalankan server hugo tidak sepenuhnya terjadi perubahan setelah saya melakukan pengaturan pada blog Hugo saya.

Apa masalahnya disini ?

Permasalahan terjadi saat saya mendapati hasil deploy yang tidak sempurna begitu saya cek perubahan yang sudah saya buat, contoh disini pada saat saya mengakses halaman About tidak sepenuhnya ter deploy seperti tulisan artikel yang saya update tidak berubah, yang semulanya saya menambahkan gambar pada halaman About namun sama sekali tidak ada saat saya mengaksesnya secara online. Lalu kenapa ini bisa terjadi ?

Apakah saat mencoba menjalankan melalui localhost ada perubahan ?

Jawabannya sama, disini saya tidak mendapati perubahannya juga. Hal ini terjadi karena saat saya menjalankan server hugo ternyata tidak ter render dengan sempurna, alhasil saat saya melakukan deploy tidak ada perubahan pada direktori /public, rendering yang tidak sempurna juga menyebabkan saat saya menjalankan server Hugo pada localhost tidak terjadi perubahan juga. Adapun pemecahan permasalahan pada kasus ini, saya dapati hanya dengan cukup merubah perintah untuk menjalankan server Hugo nya, agar perubahan terrender dengan sempurna sebelum kita melanjutkanya untuk men Deploy nya.

Perintah sebelumnya yang saya gunakan untuk menjalankan server hugo

- hugo server

Kemudian saya rubah dengan perintah ini dan berhasil

- hugo -D server

Dan permasalahan pun teratasi, namun disini penjelasan saya kuran rinci sampai saya update artikel ini. Saya masih mencari letak perbedaan penggunaan perintah hugo server dan penggunaan perintah hugo -D server next time saya akan mengupdate artikel ini untuk menjelaskan perbedaan kedua perintah tersebut untuk menjalankan server Hugo, semoga bermanfaat artikel ini ya!
