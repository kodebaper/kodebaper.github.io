Masalah error Saat Upload Artikel

Beberapa waktu lalu saya sempat menemui error pada saat akan melakukan posting artikel terbaru, seperti biasa saya update artikel di blog ini karena masih menggunakan layanan dari GitHub page dan belum ada niatan untuk pindah menggunakan layanan dari Netlifiy ataupun Forestry karena saya merasa masih nyaman untuk melakukan segala sesuatu baik update artikel terbaru penyempurnaan blog ataupun perubahan dengan cara push ke GitHub.

Oke tanpa bermaksud untuk lebih jauh keluar dari topik pembahasan pada artikel ini pembahasan saya lanjutkan, sebetulnya error yang terjadi saya sendiri kurang tau penyebabnya apa, karena sebelumnya saya tidak pernah melakukan perubahan atau aktifitas sebelum menemui error ini. Seperti biasa yang saya lakukan setiap posting artikel terbaru atau melakukan update berkala perubahan pada blog, saya selalu push ke GitHub Page dengan 5 perintah ini.

# hugo
# cd public/
# git add .
# git commit -m "pesan-commit-kamu"
# git push -u origin master

Dari perintah yang saya masukan 1 sampai 4 berjalan seperti biasa, tapi begitu saya masukan perintah push ke 5.

# git push -u origin master

Saya mendapatkan pesan error seperti ini.

error: failed to push some refs to 'https://github.com/kodebaper/kodebaper.github.io.git'
hint : Updates were rejected because the tip of your current branch is behind
hint : 'git pul ...') before pushing again.

Kurang lebih pesan error seperti di atas, hal yang selalu saya lakukan setiap menemui error adalah saya akan mulai membaca satu-persatu pesan error yang di berikan. Karena hampir setiap kita menggunakan layanan Git kemudian kita menemui permasalahan, sepengalaman saya pasti pesan error di sertakan dimana ini akan memudahkan kita untuk melakukan trace terhadap error untuk kita perbaiki. Tips lain untuk memudahkan kita dalam memahami pesan error adalah berusaha belajar bahasa inggris karena dengan memahami dan bisa bahasa inggris akan memudahkan kit dalam segala hal, terlebih sebagian besar artikel di internet menurut saya banyak sekali yang tersedia dalam bahasa inggris ataupun untuk membaca sebuah dokumentasi.

Error dapat teratasi dengan perintah ini
