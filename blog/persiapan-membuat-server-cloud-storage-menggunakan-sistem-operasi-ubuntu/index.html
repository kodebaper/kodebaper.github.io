SESION 2 Persiapan membangun server Cloud Storage

Artikel ini melanjutkan dan rangkaian dari pada bagian sesion 1 kemudian di artikel sesion 2 ini meliputi persiapan yang saya lakukan dalam praktek membuat server cloud Storage tentang apa saja yang mesti di persiapkan, adapun persiapan yang saya lakukan sebelum praktek adalah sebagai berikut:

  1. Sistem Operasi Ubuntu 16.04 Server Edition

    Saya memutuskan untuk praktek pembuatan server cloud storage dengan menggunakan sistem operasi linux Ubuntu 16.04 server edition, alasan utama saya menggunakan ubuntu lebih menitik beratkan karena pengalaman ini merupakan hal pertama yang saya lakukan dalam pembuatan server cloud storage serta setelah saya banyak mencari referensi untuk praktek ini artikel sangat banyak pembahasan mengenai proses development server cloud storage dengan sistem operasi ubuntu. Kedepan saya akan meriset untuk performa server cloud storage ini dari segi performa akan lebih cenderung unggul mana menggunakan sistem operasi ubuntu atau CentOS.

    CentOS menjadi pilihan kedua saya untuk uji coba kedepan untuk pembangunan server Cloud Storage, karena memang latihan ini tidak hanya sebatas sampai testing di localhost namun saya benar-benar akan terapkan ke dalam PC yang saya siapkan khusus sebagai server online cloud storage sebagai backup data-data saya nantinya.

  2. Virtual Machine

    Dalam pelaksanaan praktek ini saya mencobanya menggunakan virtual Machine dimana aplikasi yang saya pilih adalah Virtual Box last version saat ini, di dalam virtual Box saya menginstall ubuntu server 16.04 tersebut. Jadi urutannya seperti ini, Laptop saya menggunakan sistem operasi Mac OS > di dalam sistem operasi Mac OS saya install VritualBox > di dalam VirtualBox saya menginstall Ubuntu servernya.

  3. Mempersiapkan LAMPP

    Seperti panduan yang saya baca, saya mesti menyiapkan Lampp di dalam sistem operasi Ubuntu yang akan saya bangun server cloud storage tersebut, dimana Lampp ini include dengan Apache, MySQL, dan PHP.

  4. Aplikasi Next Cloud

    Aplikasi ini berlisensi open source sehingga saya dengan mudah bisa mendapatkanya di internet pada website resmi next cloud di https://nextcloud.com/ , Next Cloud disini nantinya aplikasi yang akan kita gunakan sebagai aplikasi graphical user interface yang kita panggil melalui browser dan lalu menempatkan data yang akan kita backup.

  5. Mempersiapkan Database

    Disini saya juga membuat sebuah database sebagai basis penyimpanan dan pengaturannya, di dalam praktek ini saya menggunakan database MySQL.

Cukup sampai di 5 poin persiapan ini dan selanjutnya mari kita mulai praktek yang akan saya bahas di Sesion 3 kelanjutan dari artikel ini.
