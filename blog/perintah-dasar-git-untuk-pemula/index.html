Yuk belajar! Perintah Dasar Git Untuk Pemula

Berikut merupakan perintah dasar dalam penggunaan Git yang cocok untuk di ketahui dan di pahami oleh seorang pemula, lebih tepatnya untuk saya pribadi karena dengan hadirnya artikel ini salah satunya sebagai tempat saya merangkum apa-apa yang sudah saya pelajari, dan untuk seorang pemula seperti saya ini sangatlah perlu. Saya sendiri sampai dengan artikel ini di posting, saya baru mengenal Git sekitar 7 bulan yang lalu.

Kemudian secara berkala artikel ini juga akan saya update dengan melanjutkan untuk pembahasan yang serupa dengan artikel ini namun dengan perintah Git dan penjelasan yang lain seiring berjalannya waktu saya menggunakan git. Daftar dibawah ini merupakan kategori perintah Git dasar yang wajib untuk di ketahui, berikut penjelasannya:

  • # Git add

    Perintah ini digunakan untuk menambahkan file baru pada repository yang dipilih.

  • # git status

    Perintah ini bisa kita gunakan untuk mengetahui status dari repository lokal yang kita buat.

  • # git commit

    Perintah ini kita gunakan untuk menyimpan perubahan yang dilakukan, tetapi tidak ada perubahan pada remote repositorynya.

  • # git push

    Perintah ini digunakan ntuk mengirimkan perubahan file setelah di commit ke remote repository.

  • # git init

    Perintah ini digunakan untuk membuat repository pada file lokal kemudian dimana nantinya ada folder .git

  • # git clone

    Perintah ini digunakan apabila kita ingin membuat Salinan pada repository lokal.

  • ## git branch

    Perintah ini digunakan jika kita ingin melihat seluruh branch yang ada pada repository.

  • # git merge

    Perintah ini digunakan untuk menggabungkan branch yang aktif dan branch yang dipilih.

  • # git log –author=namaAuthor

    Perintah ini digunakan untuk melihat komit penulis tertentu.

  • # git log –name-status

    Perintah ini digunakan untuk melihat berkas/repository yang mengalami perubahan.

  • # git log –help

    Lebih tepatnya kamu bisa mengetikan git log –help untuk mengetahui parimeter-parimeter yang lain.
