SEASON 3

Saya menggunakan Linux karena ingin Belajar

Alasan utama saya memilih sistem operasi Linux ubuntu untuk saya pasang ke notebook pertama saya adalah, saya sempat terinspirasi dari beberapa perangkat teknologi yang di bangun menggunakan sistem operasi linux, salah satunya adalah dimana berbagai macam server teknologi ternyata di bangun dan di kembangkan menggunakan sistem operasi Linux (kala itu saya baca-baca git sih). Tapi inti poin dari semua itu bukan disini, saya merasa bahwa Linux menjadi sistem operasi pilihan saya karena disamping saya melanjutkan pendidikan ke jenjang SMK dengan jurusan Teknik Komputer dan Jaringan, saya juga sama sekali belum pernah mengoperasikan yang namanya komputer atau Laptop.

Lalu apa hubungannya pilihan linux “Ubuntu” dengan jurusan sekolah yang saya ambil di jenjang SMK Teknik Komputer Jaringan dan belum pernah mengoperasikan Komputer ?

Saya menemukan gagasan setelah mencari tau seputar sistem operasi linux, bahwa nantinya dengan menggunakan Linux saya akan banyak belajar dengan sistem operasi tersebut, lalu dengan kekurangan saya yang sama sekali waktu itu tidak pernah menggunakan komputer/laptop (boro-boro make, hidupin sama matiin aja engga tau), saya masih belum mengetahui perbedaan-perbedaan antara sistem operasi yang ada. Seperti membandingkan antara Linux, Windows dan Mac OS. Jadi sulit tidaknya linux justru menghantarkan saya untuk terus belajar sampai bisa dan mulai memahami karakteristik sistem operasi Linux yang saya gunakan.

Apa setelahnya yang saya dapatkan ?

Benar saja, baik mata pelajaran di SMK mengenai pemrograman, networking, saya sangat terbantu dengan Linux ubuntu yang saya gunakan di laptop saya itu, karena banyak sekali tutorial dan pembelajaran yang memang di praktekan menggunaka sistem operasi Linux di setiap artikelnya.

Bagi kamu alasan migrasi karena apa ?

Saya berharap untuk kamu yang ingin migrasi menggunaka sistem operasi linux karena memiliki kemauan untuk belajar memperdalami disiplin ilmu tertentu, ini selain karena kamu beralasan ingin menggunakan piranti lunak yang legal ya, karena ini sudah pasti kamu dapatkan ! dan membantu negara kita juga tentunya agar citra buruk oleh negara lain yang mengecap bahwa penduduk indonesia suka menggunakan software bajakan, dan faktanya negara kita pernah menduduki peringkat teratas (lupa saya urutan berapa) bahwa negara ini termasuk negara dengan pengguna software bajakan terbesar….miris yah.

Oke kita kembali lagi

Saya ambil satu contoh mudah ya…

Kamu ingin menguasai kemampuan dalam development server ?

Atau mempelajari teknologi cloud yang saat ini semakin trend dan di butuhkan tenaga ahli di bidang server cloud.....?

Jika seperti itu, ini merupakan langkah terbaik untuk kamu untuk memutuskan mulai fokus dan menekuni untuk belajar dengan sistem operasi linux, karena dari sini dua pertanyaan diatas akan terjawab.

Atau malah kamu ingin menjadi seorang Security Engineer ahli, maka migrasi ke sistem operasi linux, dan mempelajari dasar-dasar sistem operasi linux kemudian pelajari lebih lanjut merupakan solusinya.

Saya menarik kesimpulan disini !!! Bahwa kamu yang ingin mempelajari linux dan memilih distro yang memang kamu anggap cocok menjadi distro pilihan pertama kamu untuk migrasi ke sistem operasi linux, tanamkan niat yang kuat untuk belajar sungguh-sungguh tentang ilmu yang kamu akan pelajari. Lalu pada intinya kamu memiliki tujuan yang kuat serta goal kedepan mengapa sistem linux yang kamu pilih untuk migrasi.
