Catatan Belajar Pemrograman, Linux Server Enthusiast & Ghibah Seputar Dunia IT

Pekerjaan selanjutnya setelah kita membuat blog dengan static site Hugo lalu memasang template dan mulai mengisi dengan konten, hal yang tak kalah penting adalah menghadirkan fitur komentar di setiap artikel pada blog. Ini akan sangat penting agar tulisan kita layaknya sebuah percakapan yang membutuhkan lawan bicara untuk menanggapi percakapan kita yaitu melalui kolom komentar cara lawan bicara kita menanggapinya.

Pada kesempatan kali ini saya akan membahas konfigurasi Disqus comment yang di hadirkan pada blog hugo saya, pada dasarnya di website resmi hugo pada bagian dokumentasinya sudah di jelaskan dengan sangat rinci untuk bagaimana mengkonfigurasi disqus comment di hugo.

Untuk itu saya berpatokan dari dokumentasi Hugo dan memang sudah ada konfigurasi khusus ke bagian template yang saya download dan saya pasang di blog saya ini sehingga saya tinggal melakukan kofigurasi dan meregistrasi blog saya di website resminya Disqus lalu saya tinggal menambahkan disqus shortname di file config.toml hugo saya.

  1. Pertama disini saya anggap sudah mendaftarkan account di website resmi Disqus.

  2. Setelah itu kita akan di hadapkan dengan tampilan seperti ini, pilih saja GET STARTED

  3. Kemudian selanjutnya kita akan di hadapkan pada tampilan seperti di bawah ini, pilih saja I want to install Disqus on my site

  4. Pada langkah ini kita akan di hadapkan dengan tampilan seperti ini, lalu isi saja sesuai dengan yang tertera kemudian pilih Creat Site

  5. Langkah berikutnya kita mendapati tampilan seperti di bawah ini, namun kita akan menginstallnya secara manual dengan memasukan code yang akan diberikan oleh Disqus nantinya untuk kemudian kita pasang di blog Hugo kita, langkahnya cukup mudah scroll saja sampai mentok kebawah lalu ada petunjuk untuk memasang disqus secara manual.

    Pada bagian paling bawah ada form seperti ini untuk menginstall disqus secara manual, lalu klik saja petunjuk dibawah ini.

  6. Tampilan berikutnya setelah kita klik akan seperti gambar dibawah ini, scrol saja kebawah dan bisa mengikuti petunjuk yang sudah di jelaskan pada videonya, saya tidak akan membahasnya disini karena memang di videonya sudah di jelaskan tinggal menontonnya saja.

  7. Kita akan masuk ke tahap akhir dimana kita tinggal satu langkah lagi untuk mengakhiri proses konfigurasi di website Disqus nya itu sendiri. Kemudian kita memasuki tampilan seperti pada gambar di bawah ini.

    Kemudian isi saja sesuai apa yang tertera dari petunjuk di masing-masing formnya.

    Langkah terakhir kita akan memasukan Disqus shortname nya ke pengaturan file config.toml di dalam file hugo kita.

  8. Isi saja dengan diqus shortname kita, pada file config.toml seperti gambar di bawah ini.

    Ini merupakan langkah terakhir kita melakukan konfigurasi untuk memasang komentar Disqus pada blog hugo, semoga artikel ini bermanfaat, selamat belajar!!!
