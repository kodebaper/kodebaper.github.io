Yakin mau belajar KOTLIN?


Ini merupakan artikel perdana saya, dimana pada artikel ini merupakan catatan pertama kalinya saya belajar pemrograman mobile menggunakan Kotlin, lebih tepatnya ini bukan merupakan catatan belajar sih tapi agenda paling awal yang saya susun untuk mempelajari bahasa pemrograman dengan menggunakan Kotlin, dimana rencananya saya akan fokus belajar mobile programming menggunaka bahasa pemrograman Kotlin. Lalu disini saua akan menguraikan beberapa hal yang mendasari saya belajar Kotlin diantaranya yaitu :

  1. Prospek Kedepan yang bagus terhadap profesi menjadi seorang Programmer aplikasi mobile.

  2. Saya tertarik untuk membuat aplikasi mobile yang dapat bermanfaat untuk orang lain.

  3. Kotlin bahasa pemrograman pertama yang membuat ketertarikan saya akan belajar pemrograman untuk mobile itu muncul.

  4. ingin merasakan keluar dari rutinitas yang selalu bekerja di bidang Networking.

Adapun hal yang saya lakukan untuk mulai mempelajari Kotlin meliputi :

  1. Saya banyak membaca tutorial dasar-dasar pemrograman menggunaka Kotlin untuk pemula

  2. Saya langsung memasang persenjataan untuk praktek pemrograman Kotlin menggunaka aplikasi Android Studio

  3. Mendownload sebanyak mungkin dasar-dasar kotlin berupa ebook ataupun artike berbentuk PDF di internet

  4. Mulai memilih kelas kursus online untuk mengambil kursus online belajar bahasa pemrograman menggunakan Kotlin

Itu dia empat hal yang mendasari dan empat hal yang saya lakukan untuk mulai berkenalan dengan Kotlin. Selamat datang Kotlin untuk perkenalan kita semenjak 1 bulan yang lalu hingga hari ini, dimana saat ini saya sedang fokus dan konsentrasi penuh untuk belajar bahasa pemrograman yang satu ini.
