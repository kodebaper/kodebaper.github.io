Pahami perintah dasar Terminal ini jika kamu seorang Pemula

Ini merupakan sesion terkahir yang saya bahas mulai dari sesion 1 sampai dengan sesion 6 ini dimana saya membahas mengenai perjalanan saya menggunakan sistem Operasi Linux Ubuntu untuk pertama kalinya dan saya selipkan juga sebagai langkah awal untuk kamu yang memutuskan untuk migrasi ke sistem operasi Linux. Pada pembahasan langkah ke 6 ini merupakan pembahasan paling dasar dimana seorang pemula, mulai mengenal apa itu aplikasi Terminal pada Ubuntu dan menggunakan perintah-perintah di terminal.

Saya ingat betul waktu itu hanya beberapa perintah dasar terminal yang saya mulai coba untuk mulai mengenali apa itu terminal Ubuntu dan fungsinya, perlu di ketahu aplikasi terminal pada Linux merupakan tool berbasis text yg digunakan untuk berinteraksi dengan komputer, dimana sebelum adanya kita kenal dengan yang namanya Graphical User Interfaces lebih dulu berinteraksi dengan komputer menggunakan sistem berbasis text.

Nah berikut merupakan perintah dasar yang bisa menjadi panduan kamu memulai belajar menggunakan terminal, dimana ini perintah dasar pertama kali yang saya gunakan pada Linux ubuntu saya.

1. sudo su  [untuk berpindah dari user biasa ke root user]
2. ls [list, untuk melihat seluruh isi file pada sebuah direktori]
3. mkdir [untuk membuat sebuah direktori baru]
4. cd [untuk berpindah dari satu direktori ke direktori yang lain]
5. rm [untuk menghapus sebuah file]
6. rm -r [untuk menghapus sebuah folder]
7. pwd [untuk mengetahui direktori saat ini kita berada]
8. cat nama file [untuk membaca isi dari file]
9. ls -l [untuk melihat full isi dari direktori dengan permission, nama user, grup. dsb]
10. sudo apt-get update [untuk melakukan pembaharuan package secara berkala]

Ini merupakan perintah dasar terminal yang paling umum sering di gunakan dan merupaka perintah terminal pertama kali saya mencobanya di ubuntu pada waktu itu, dan terus saya biasakan berulang-ulang perintah tersebut sampai saya memahami keseluruhan fungsinya masing-masing. Semoga ini bermanfaat ya, selamat belajar !
