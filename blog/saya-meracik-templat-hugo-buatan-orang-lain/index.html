0. Tips menggunakan template karya orang lain

Pada pembahasan artikel kali ini, saya akan membahas detail template yang di pakai pada blog saya saat ini. Namun sebelum membahasnya saya sempatkan juga untuk membahas tips menggunakan template hasil karya orang lain agar kita memahami setiap bagian dan tata letaknya. Dimana hal yang saya lakukan setelah mendownload template hasil karya orang lain saya akan membedah dan bila perlu mencatat satu persatu isi dari file yang ada di template tersebut.

Kemudian saya akan mulai membaca dan memahami isi kode-kode yang tersusun di dalamnya, namun hal paling awal yang saya lakukan saat itu adalah mencatat keseluruhan isi file di dalamnya, tampak seperti gambar di bawah ini saya mencatat file yg terbentuk di dalam satu template yang saya gunakan saat ini.

Gambar di atas merupakan isi dari seluruh file template yang saya gunakan lalu saya gambarkan di selembar kertas agar mudah untuk di pelajari dan mencoba membedah secara berurutan sesuai urutan susunan file yang tampak seperti di gambar.

Mari kita lanjut ke materi pembahasan sesuai judul dimana ada beberapa point yang akan saya bahas di antaranya yaitu :


1. Apa nama template yang di pakai ?

2. Kenapa saya tidak membuat template sendiri ?

3. Yang saya lakukan setelah menemukan template yang di inginkan.

Oke, mari kita bahas 3 point diatas untuk lebih jelasnya mengenai template ini.

1. Apa nama template yang di pakai ?

Template yang saya pakai untuk blog hugo saat ini adalah template bernama Hago theme yang saya dapatkan secara gratis DI SINI atau bisa download dengan menuju ke GitHub orang di balik pembuat template DI SINI. Saya suka dengan template ini karena simple, mengingatkan kebiasaan saya senang membaca artikel di situs terkenal Medium, simple tapi enak buat di baca setiap isi dari konten di dalamnya karena menurut saya tata letak dari setiap komponen yang ada di Medium itu tertata rapi dan tidak ribet. Itulah yang saya rasakan begitu menemukan template ini. Fitur yang di hadirkan oleh pembuat untuk template ini diantaranya :

- Responsive web
- Tags Page
- Categories Page
- Google Analytics
- Comments
- jQuery 2.2.4
- Bootstrap 4.4.1
- Awsome Font 5.8.2

Apakah cukup sampai disini fiturnya ? tentu tidak ya ada beberapa yang mesti kita sempurnakan dan di sesuaikan dengan keinginan kita. Namanya saja template bikinan orang lain jadi kita menyesuaikan dengan si pembuatnya lalu untuk penyempurnaan tinggal kita tambahkan.

Keuntungan yang saya dapatkan dengan menggunakan template hasil karya orang lain yaitu:

  • Hemat waktu karena saya tidak membuatnya dari nol

  • Bagi pemula seperti saya, sangat terbantu untuk memiliki template seperti ini dimana jika saya membuat dari nol akan lebih memakan waktu.

  • Saya jadi memiliki ilmu baru, seperti yang saya katakan karena saya masih sangat awam dengan dunia programming maka saya akan terbawa untuk mau mempelajari apa saja yang di butuhkan untuk membuat sebuah template seperti yang saya pakai ini.

Kelemahan dengan menggunakan template hasil karya milik orang lain diantaranya:

  • Ada bagian dari fitur pada template yang tidak sesuai keinginan, ini sudah pasti karena terkadang sekalipun tampilan depan dari template tersebut sudah sesuai dengan yang kita mau tapi ada fitur lain yang ternyata jauh dari ekspektasi kita.

  • Jauh lebih sulit untuk mempelajari seluruh komponen yang tersusun di dalam template itu karena bukan kita yang merancangnya dari awal.

  • Rasa kepuasan tersendiri yang kurang menurut saya, karena keseluruhan baik isi konten dari blog ataupun template tidak sepenuhnya kita yang menciptakan.

2. Kenapa saya tidak membuat template sendiri ?

sebenarnya sebelum saya menggunakan template ini, saya sudah sempat membuat template sendiri dengan mengikuti panduan membuat template dari nol untuk Hugo dimana panduan ini saya dapat dari blog yang sama-sama di bangun menggunakan hugo namun blog ini saya rasa sangat bagus dan selalu memotivasi saya untuk memiliki blog sekelas blog ini yaitu [Petani Kode]. Setelah beberapa pertimbangan saya merasa belum ada kesempatan untuk melanjutkan pembuatan template yang pernah saya buat lalu memasangnya ke blog ini karena saya saat ini sedang fokus untuk belajar mobile programming dengan menggunakan bahasa pemrograman Kotlin.

Saya berfikir akan memecah fokus jika saya membangun template sendiri namun saat ini lagi semangat-semangatnya belajar Kotlin, akhirnya saya putuskan untuk menggunakan template hasil karya orang lain sementara dimana saatnya nanti memungkinkan akan saya update blog saya ini untuk menggunakan template hasil buatan sendiri, dimana saya tinggal menyempurnakan template yang sempat saya buat sebelumya.

Hal ini juga lebih ke efisiensi waktu dimana semakin cepat blog yang saya bangun dengan hugo ini jadi akan semakin cepat saya mendokumentasikan setiap ilmu baru yang saya pelajar dan menuangkannya dalam bentuk tulisan di blog ini.

3. Yang saya lakukan setelah menemukan template yang di inginkan.

Yang perlu di pahami setelah menemukan template sesuai dengan keinginan yaitu tidak berharap sempurna untuk keseluruhan yang ada di dalam template tersebut tentunya ya, karena pasti kita membutuhkan perubahan lain dari template yang sudah kita download atau paling tidak menyempurnakan fitur karena template yang orang lain buat belum tentu keseluruhan sesuai ekspektasi. Setelah point tips seperti di awal artikel ini saya bahas dimana saya sudah paham dengan keseluruhan isi file yang menciptakan templat di blog ini, yang saya lakukan setelah menemukan template yang di inginkan adalah saya mulai pelan-pelan untuk merubah beberapa bagian, seperti nampak saat ini bisa di rasakan sudah sedikit ada perubahan tampilan template original dari penciptanya hingga saat ini saya pakai. Oh ia tidak lupa saya juga menambahkan beberapa bagian script untuk kebutuhan SEO di beberapa file template ini.

Semoga artikel ini membantu, selamat belajar!
