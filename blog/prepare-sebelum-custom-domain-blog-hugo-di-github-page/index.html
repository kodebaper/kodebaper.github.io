Prepare Sebelum Custom Domain Blog Hugo Di Github Page

Berikut merupakan hal-hal yang perlu di siapkan sebelum melakukan custom domain terhadap Blog Hugo ini, dimana saya mealkukan perubahan atau custom domain ini dengan menggunakan layanan dari Github Page, adapun untuk persiapannya sebagai berikut.

  1. Menyiapkan domain

    Pastikan kamu sudah membeli domain ya, jika kamu menggunakan domain yang berbayar. disini saya mendokumentasikan persiapan saya sebelum melakukan custom domain di github page dengan menggunakan domain berbayar, saya membeli domain .com di Domainesia , ini juga nantinya akan berpengaruh ke setting DNS Management yang nantinya mesti kita rubah dimana pada kasus ini saya merubah Nameserver default saya dari Domainesia menjadi Nameserver yang di rekomendasikan cloudflare karena saya juga menggunakan cloudflare untuk DNS management nya.

  2. Baca Dokumentasi

    Ini sangat penting ya, perlu di pahami sebelum melakukan custom domain, karena dalam tahap ini kita akan melakukan beberapa perubahan yang mungkin belum kamu pahami betul seperti, pemahaman management DNS, apa itu merubah Nameserver untuk mengarahkan domain awal kamu ke domain yang baru, kamu mungkin belum begitu paham untuk pengaturan CNAME, ataupun aktifasi SSL.

    Dokumentasi disini saya sertakan untuk menjadi modal panduan kamu belajar ya, perlu di pahami, cara yang saya gunakan untuk urutannya seperti ini kurang lebih agar kamu memahami urutannya,

    [Saya membeli domain ---> setelah domain saya telah di aktifkan oleh Domainesia --> saya menggunakan Cloudflare untuk management DNS lalu saya ADD DOMAIN di Cloudlfare ---> Saya melakukan pengaturan Nameserver perubahan dari Cloudflare ke Dashboard Domainesia saya ---> lalu saya custom domain di github page dari repository utama]

    kurang lebih urutannya seperti ini, lalu saya akan jabarkan lebih rinci lagi langkah-langkah custom domain di github page untuk blog hugo saya ini di lain kesempatan.

    Berikut dokumentasi yang bisa kamu baca untuk menjadi panduan kamu.

    Panduan pertama:
https://docs.github.com/en/github/working-with-github-pages/configuring-a-custom-domain-for-your-github-pages-site

Panduan kedua:
https://docs.github.com/en/github/working-with-github-pages/managing-a-custom-domain-for-your-github-pages-site

    Dua modal dokumentasi ini saya rasa sudah cukup, karena sangat rinci dan lengkap sebagai panduan untuk melakukan custom domain kamu di github page.

  3. Sudah mendaftarkan Account di layanan CLoudflare

    Sebaiknya kamu sudah terlebih dahulu membuat account di Cloudflare, sangat mudah untuk membuatnya karena hanya memerlukan email untuk proses pendaftaran dan altivasi account nantinya. Kemudian kamu disini menggunakan layanan free yang di sediakan cloudflare seperti yang saya lakukan, namun untuk paket berbayar jika kamu menginginkannya juga tersedia. Persiapan ini penting untuk mempercepat proses custom domain karena setelah domain yang kita beli aktif kemudian kita bisa langsung mendaftarkan domain kita tersebut di layanan Cloudflare.

Sekian untuk artikel kali ini, saya sangat merekomendasikan persiapan ini untuk di penuhi guna memudahkan proses belajar kita untuk bereksperimen dengan Hugo dan GitHub Page. Semoga artikel ini bermanfaat, selamat belajar !
