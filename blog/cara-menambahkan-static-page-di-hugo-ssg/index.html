Cara Menambahkan Static Page Di Hugo SSG

Site plain yang saya lakukan setelah blog sudah berjalan, saya melakukan penyempurnaan terhadap fitur dan performa terhadap blog. Namun ada hal yang saya rasa masih sangat sulit untuk saya realisasikan adalah mengoptimalkan performa blog yang lebih cepat dan familiar di skala search engine/optimasi SEO terhadap blog.

Sebenarnya ada hal sulit lain yang sampai sekarang masih saya coba pelajari, yaitu merapihkan code yang sudah saya tulis, sudah sesuai dengan aturan penulisan code yang benar ataupun efektif tidaknya code-code yang di tulis untuk blog ini. Namun tidak berakhir sampai disini karena dua kendala kesulitan yang saya hadapi saat ini tidak menurunkan semangat saya belajar, sembari mempelajarinya saya melakukan langkah lain untuk optimasi penyempurnaan blog ini tentunya.

Dalam hal ini yang saya lakukan adalam melakukan optimasi terhadap fitur, optimasi yang saya lakukan disini meliputi 2 fitur diantarnya ;

  1. Memberikan tambahan page baru untuk memperkaya fitur, yaitu about page, contact page dan FaQ page, saat ini baru terealisasi untuk about page karena kesibukan pekerjaan dan laih hal sehingga saya belum bisa melengkapi keseluruhan page yang mestinya saya buat.
  1. Menambahkan menu Roadmap untuk mengumpulkan seluruh fitur dan menu yang ada di dalam blog. Dua fitur yang saya lakukan ini cukup mudah untuk di lakukan, terutama fitur pertama untuk menambahkan page baru di Hugo.

Hal menarik justru saya dapat pelajaran banyak, disini saya menambahkan fitur Roadmap dimana sama-sama page namun lebih komplek. Ceritanya disini saya menambahkan menu Roadmap untuk mengumpulkan seluruh menu dan fitur yang ada di blog, initnya semua menu blog bisa di akses di satu page yaitu page pada Roadmap.

Apa alasan utama membuat menu Roadmap ?

Roadmap disini saya terinspirasi dari dokumentasi dari sebuah pelajaran/program, dimana saat orang ingin mempelajari sesuatu dan membuka roadmap, maka di dalam Roadmap akan memberikan seluruh isi panduan dari sebuah program yang di urutkan langkah-langkahnya secara sistematis dan terstruktur.

Maka dari pengalaman sebuah Roadmap yang mengagumkan, saya akhirnya memiliki ide menghadirkan Roadmap di blog ini dan saya merencanakan untuk menyempurnakan fitur ini untuk memudahkan dalam hal menjangkau seluruh isi dari blog saya ini.

Menyenangkan memang semakin mendalami static site generator dari Hugo ini, berawal dari nol saya tidak mengetahui sama sekali apa itu ilmu programming akhirnya saya mulai mengenal apa it HTML, CSS, JAVASCRIPT, GOLANG, Bootstrap dan beberapa hal lain yang berkaitan dengan hal-hal yang berbau dengan dunia web programming.

Sederhana memang bagi orang yang ahli di bidang website untuk memahami apa itu static site generator, namun bagi saya hugo merupakan langkah yang sangat baik untuk memulai sebuah belajar. Belajar dunia website dan programming sederhana untuk menambah batu loncatan yang lebih kompleks lagi kedepannya mengenai apa itu dunia programming.
