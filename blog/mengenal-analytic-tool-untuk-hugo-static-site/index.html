Mengenal Analytic Tool Untuk Hugo Static Site

Sudah dari dua hari yang lalu sampai pada Artikel ini saya posting, saya sedang sangat fokus untuk memahami persenjataan baru untuk kebutuhan SEO blog saya pribadi, yaitu JavaScript Analytic dan Google Tag Manager. Kedua produk ini dari google dan sejauh ini saya cukup mudah untuk memahami serta melakukan proses penerapannya pada blog saya. Namun banyaknya fitur yang di berikan oleh kedua tools untuk keperluan SEO / Marketing dari google ini tidak cukup satu atau dua hari untuk saya bisa memahami isi fitur di dalamnya.

Sebelumnya saya sudah memasang Google Analytic dan Google Search Console serta Bing Webmaster untuk kebutuhan SEO blog saya ini, namun apakah efektif dengan hadirnya berbagai macam tool untuk SEO pada blog ?
Sejauh ini dari yang saya baca-baca tidak ada masalah, karena sekalipun membenamkan script di dalam root file HTML blog hugo, script untuk mejalankan tools-tools Analytic ini berkerja masing-masing tanpa memunculkan bug atau memperlambat load pada blog. Sejauh ini seperti itu yang saya rasa.üßê

Saya selalu memantaunya setiap perkembangan dari blog ini menggunakan tools-tools analytic yang saya pasang, sangat membantu bahkan sampai ke tingkat analisa keyword yang mesti saya buat, artikel yang relevan untuk mudah di temukan di mesin pencari. Saya juga percaya bahwa blog yang ramai di kunjungi oleh pembaca selain menyajikan konten-konten yang berkualitas juga memerlukan jam terbang waktu serta konsisten dalam meng-update isi artikel di dalamnya.
