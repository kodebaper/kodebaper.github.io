Solved : Fatal Not a Git Repository or Any of the Parent Directories.

Bagi saya untuk mendokumentasikan setiap error yang di temui dalam setiap pembelajaran sangatlah penting, terlebih saya adalah seorang pemula untuk belajar membangun blog dengan menggunakan static site Hugo, kemudian belajar programming dengan Kotlin maka dari itu mendokumentasikan setiap pembelajaran dengan menuangkan dalam sebuah tulisan di blog merupakan suatu keharusan untuk saya sendiri dimana ini akan bermanfaat kedepannya.

Ini adalah dokumentasi kesekian kalinya, dimana pada kasus ini saya mengalami kendala dimana template yang ingin saya clone dari hasil template karya orang lain tidak bisa di pasang melalui git ke blog saya. Berikut pesan error yang di tampilkan begitu saya menjalankan perintah untuk memasang templatenya di blog, ini adalah perintah untuk memasang template yang ingin saya pasang di blog pribadi saya.

git submodule add https://github.com/gyuha/hago.git hago

Ini adalah pesan error yang di tampilkan setelah mengeksekusi perintah di atas

fatal: Not a git repository (or any of the parent directories): .git

Lalu bagaimana untuk memperbaiki error tersebut ? Berikut adalah solusinya untuk mengatasi permasalahan dengan pesan error seperti di atas.

git init

Kemudian coba, jalankan kembali baris perintah diatas untuk mencoba memasang kembali templatnya, seharusnya sudah bisa. Permasalahan yang mungkin sangat sepele untuk penyelesaian yang singkat bagi sesorang yang sudah familiar dengan perintah git, namun tidak untuk seorang pemula seperti saya.

Berikut adalah penjelasan dari perintah # git init dimana perintah ini digunakan untuk membuat repository pada file lokal yang nantinya ada folder .git. semoga artikel ini bermanfaat, selamat Belajar!
