Belajar Mendokumentasikan Kode, Error & Cerita Secangkir Kopi

Buat blog dari nol cukup dipersenjatai aplikasi console Terminal dan Text Editor

Saya mengawali dunia Blogging dari CMS…..

Ngobrol produktif kali ini saya akan bercerita sedikit tentang pertama kalinya saya membuat website sendiri lebih tepatnya blog sebenarnya hingga dapat di akses di internet, karena ternyata tidak semua website di buat dengan kerumitan yang selalu di bayangkan oleh seorang yang masih awam dalam dunia programming khususnya web progrmming, orang awamnya itu saya sendiri tentunya. Saya sendiri sempat membayangkan untuk membuat website itu sulit dan butuh keahlian yang baik dalam bidang programming, lebih jauhnya saya membayangkan ingin memiliki blog tempat saya meletakan tulisan-tulisan saya di internet dimana seluruh isi dari blog baik proses pembuatan hingga mengisi konten saya harus paham seluruhnya.

Sempat menggunakan CMS seperti Blogger, Wordpress dan Joomla namun saya merasa seiring berjalannya waktu bahwa blog yang saya kelola saya merasa tidak memilik kendali penuh terhadap blog saya sendiri, saya kurang memahami blog secara keseluruhan. Karena dengan menggunakan CMS saya hanya memsang segala fitur di dalam blog cukup dengan apa yang sudah disediakan di dalam CMS tersebut kalau tidak ada, saya tinggal menambahkan plugin yang dengan mudah saya dapat download di internet, ya sesimple itu. Tidak sampai disitu untuk urusan template juga kita tinggal memilih di internet dan mendownloadnya secara gratis lalu memasangnya di blog CMS kita.

Saya merasa bosan dengan hal itu dan tidak sesuai keinginan…..

Akhirnya saya mulai memikirkan bagaimana caranya bisa ngeblog dengan suasana yang sedikit bereda, walaupun belum ada basic ilmu di bidang penguasaan bahasa pemrograman untuk membangun website/blog. Namun hal terpenting ada pada point dimana saya menginginkan bisa memahami keseluruhan isi blog yang saya kelola bukan hanya terhadap konten yang saya isi di blog tersebut. Saya mengerti proses pembuatan blog, saya mengerti proses deploy website yang dari awalnya offline sampai bisa online, saya paham dimana tempat blog saya bisa di onlinekan sehingga bisa di akses di internet, saya paham keseluruhan intinya blog yang saya kelola dengan kesimpulan saya mesti membangun sendiri dari Nol.

Nah ini maksudnya saya…..

Saya menemukan jawaban atas semua yang saya inginkan tadi, saya menemukan sebuah Framework yang di klaim sebagai salah satu jenis Framework tercepat di dunia. Yaitu membangun blog dengan menggunakan static site generator Hugo dimana lewat banyak referensi, saya mulai membangun sebuah blog yang saya bangun sendiri menggunakan Hugo dengan bermodalkan dokumentasi dari website resmi hugo serta tutorial yang banyak bertebaran di internet. Hanya itu modal saya pertama kali berkenalan dengan salah satu static site generator yang saya pakai saat ini untuk blog pribadi saya.

Saya menemukan komposisi awal untuk blog ini…..

Saya menemukan komposisi untuk meracik blog ini hingga sekarang saya aktif menulis di blog ini, dimana komposisi blog pertama kali saya bangun diantaranya:

  • Blog menggunakan static site generator Hugo

  • Saya menggunakan aplikasi console Terminal dan text editor Sublime.

  • GitHub page sebagai sarana saya men-Deploy nya.

  • Domain awal sebelum saya custom [kodebaper.github.io]

  • Custom domain saya lakukan tetap menggunakan layanan GitHub Page menjadi kodebaper.com.

  • Bermodal SSH Key yang di setting ke GitHub untuk setiap kali saya push perubahan blog.

  • Membeli domain .com di layanan penyedia domain dari Domainesia [dari dulu saya langganan disini].

  • Konfigurasi DNS dengan memanfaatkan layanan Cloudflare.

  • Mempercayakan Analytic blog dengan layanan Google Analytic.

Sudah sampai disini? …..

Tentu tidak masih ada beberapa hal yang saya lakukan sebenarnya, mulai dari belajar merapikan kode, mempelajari teknik SEO untuk blog ini, dan masih banyak lagi untuk mulai menyempurnakan blog buatan saya sebagai orang awam supaya lebih enak di lihat dan di baca kontennya.

Ada keinginan kedepan untuk Blog ini? …..

Ada, banyak malah keinginan yang masih terbesit di pikiran saya. Saya ingin belajar bahasa Golang karena memang hugo ini di bangun dengan bahasa golang kalau saya tidak salah ingat dari beberapa informasi yang saya baca, dengan mengerti bahasa Pemrograman golang saya akan semakin mengerti hugo lebih mendalam. Saya ingin pindah juga dari rutinitas yang selalu mengharuskan saya untuk push ke github dan mencoba untuk menggunakan layanan dari Netlify atau kalau tidak Forestry namun masih bingung saat ini lebih bagusan yang mana dan cocok untuk blog Hugo saya. Saya juga merasa ingin berpindah layanan dari GitHub Page untuk mencoba mendalami dan menggunaka layanan dari Amazone Web Services untuk blog saya ini. Saya juga ingin membuat template sendiri di desain dari nol supaya enak di baca artikelnya dari perangkat mobile, dsb.

Cukup banyak bukan ?….

Dan masih banyak lagi rencana kedepan! itu semua merupakan bagian dari proses pembelajaran yang sedang saya nikmati setiap alur dan tahapannya. Semoga ngobrol produktif ini bukan hanya sebatas Es Teh….selamat belajar!!!
